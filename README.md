1. Open `dd.html` in your web browser
2. Press any key on your keyboard to start the game
3. Watch carefully as the game flashes a sequence of colored buttons
4. Repeat the sequence by clicking the buttons in the same order
5. If you succeed, the game will add one more color to the sequence
6. Keep playing and try to achieve the highest level possible!
7. If you make a mistake, the game will display your score and you can restart by pressing any key
